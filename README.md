# Academic Reports in R and Numerical Analysis

This repository contains selected academic projects completed during university coursework.  
Most reports were prepared in **R Markdown** and rendered to **PDF**, with accompanying data files used in the analysis.  
It also includes a numerical analysis project prepared in **Jupyter Notebook**.

## Repository structure

Most project folders contain:

- `report.Rmd` – source file written in R Markdown
- `report.pdf` – rendered final report
- `data/` – files used in the analysis

The numerical analysis project **Numerical Integration Benchmarks (ENG)** includes:

- `notebook.ipynb` – Jupyter Notebook source file
- `notebook.pdf` – rendered final Notebook
- `figures/` – image files or plots generated during the analysis

## Included projects

The repository currently includes the following topics:

- **Poisson Regression** *(PL)*
- **Logistic Regression** *(PL)*
- **Maximum Likelihood Estimators** *(PL)*
- **Simple Linear Regression** *(PL)*
- **Multiple Linear Regression** *(PL)*
- **Nonparametric Two-Sample Tests** *(PL)*
- **Numerical Integration Benchmarks** *(ENG)*

## Main areas covered

The projects showcase methods and techniques related to:

- statistical modeling and inference
- regression analysis
- model diagnostics and selection
- nonparametric methods
- numerical analysis

## Tools and technologies

The projects were prepared using R, R Markdown and selected tools for statistical analysis and visualization, with Python / Jupyter Notebook used in numerical analysis.
